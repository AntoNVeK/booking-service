Это проект по реализации системы бронирования комнат в отеле на Java.

Доступные функции:
##### Гости
* Получить гостя по идентификатору
* Добавить гостя
* Редактирование гостя

##### Номера
* Получить номер по идентификатору
* Добавить номер
* Редактировать номер
* Удалить номер

##### Брони
* Получить бронь по идентификатору
* Поиск свободных номеров на дату и время
* Создание новой брони
* Редактирование брони
* Отмена брони

При работе с добавлением и обновлением брони использовал блокировку на уровне Postgeres на одной комнате при помощи for update

Оптимизация запросов с помощью limit, индексов, batch-update.

* Язык разработки Java
* Сборка проходит под Java 21
* Используется framework Spring Boot
* Система сборки - Gradle
* Для работы с базой данных NamedParameterJdbcTemplate
* База данных - PostgreSQL (поднимаю через докер)
* Миграции пишутся с помощью библиотеки Liquibase и располагаются в `src/main/resources/liquibase`
* Swagger-ui
* JUnit-тесты
* Метрики



В этой системе базово предствалены всего 3 сущности: Гость, Номер, Бронь.

### Модель данных
У сущностей следующие свойства:
##### Гость
* Идентификатор
* Фамилия
* Имя
* Отчество
* Дата рождения
* Номер телефона

##### Номер
* Идентификатор
* Этаж
* Номер
* Количестов мест

##### Бронь
* Идентификатор
* Дата и время начала брони
* Дата и время окончания брони
* Гости
* Номер
